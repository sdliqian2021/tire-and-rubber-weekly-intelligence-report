# Tire and Rubber Weekly Intelligence Report

This folder is a standalone GitHub Pages site containing only public, approved reports.

## Publish it on GitHub Pages

1. Create a public GitHub repository named `tire-and-rubber-weekly-intelligence-report` under the `sdliqian2021` account.
2. Copy the contents of this folder to the repository root.
3. In the repository, open **Settings > Pages**.
4. Select **Deploy from a branch**, then choose **main** and **/(root)**.
5. Save and wait for the Pages deployment to finish.

The site URL will be:

`https://sdliqian2021.github.io/tire-and-rubber-weekly-intelligence-report/`

## Add a future report

1. Copy only the approved, finalized Markdown report into `reports/`.
2. Give it a stable week-based filename and YAML front matter with a permalink.
3. Add its link to both **Latest report** and **Report archive** in `index.md`.
4. Commit and push the change.

Do not publish internal files such as `stories.csv`, `approval.md`, research notes, source strategies, or agent instructions.
